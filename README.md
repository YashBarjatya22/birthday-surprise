# 🎂 16th Birthday Surprise Website

A visually stunning, interactive, and emotional 16th birthday website created for my little sister. Features a fully custom UI, photo carousel, background music (BTS - Magic Shop), interactive mini-games, and a digital cake!

## 🚀 How to Run Locally

If you downloaded this repository and want to run it on your own computer:
1. Open a terminal in this folder.
2. Run the server using Java:
   ```bash
   javac Server.java
   java Server
   ```
3. Open your browser and go to `http://localhost:8080`.

## 🌐 How to Host on Render
1. Connect this repository to Render as a **Static Site**.
2. Set the Publish Directory to: `public`
3. Hit Deploy!
